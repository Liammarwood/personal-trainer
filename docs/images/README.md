## Image Guidelines

### Format
- **PNG**: Screenshots, UI elements
- **JPG**: Photos, complex images
- **GIF**: Animations, workflows
- **SVG**: Icons, diagrams (optional)

### Size
- Max width: 1200px
- Optimize for web (compress)
- Use descriptive filenames
- Keep file sizes under 500KB

### Naming Convention
- Use lowercase
- Separate words with hyphens
- Be descriptive
- Examples:
  - `main-interface.png`
  - `workout-tracking-example.png`
  - `settings-audio-options.png`

### Accessibility
- Include alt text in Markdown
- High contrast for visibility
- Clear, focused screenshots
- Annotate important areas

## Image Processing Tools

- **Screenshots**: Built-in OS tools, ShareX (Windows), Skitch (Mac)
- **Editing**: GIMP, Paint.NET, Photoshop
- **Compression**: TinyPNG, ImageOptim, Squoosh
- **Annotation**: Snagit, Skitch, Greenshot
